🧪 Projets Personnels

🎲 KELVINC – Jeu web “Des inconnus nous jugent”

Développement d’une application web inspirée du concept de Squeezie, permettant à deux groupes de personnes ne se connaissant que peu de s’évaluer anonymement via des questionnaires sur des traits de personnalité (ex. : “Qui est le plus drôle ?”).

  • Ajout dynamique des participants depuis l’interface (aucun code à modifier)
  • Formulaires personnalisés par groupe et par question
  • Calcul automatique des résultats (classements par nombre de “5” reçus)
  • Affichage du classement et du gagnant pour chaque question

Projet fullstack léger réalisé en Python (Flask) + HTML/CSS, avec stockage des données en JSON. Pensé pour être utilisable immédiatement via un simple lien, sans base de données ni configuration complexe.
